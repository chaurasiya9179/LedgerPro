# 💸 Ledger-app

A powerful and user-friendly **Ledger Management System** designed to track transactions, manage balances, and handle financial records in real-time.

## 🚀 Features
* **Real-time Updates:** Powered by Supabase for instant data synchronization.
* **Modern UI:** Built with React.js for a smooth and responsive user experience.
* **Secure Authentication:** User login and data protection via Supabase Auth.
* **Transaction History:** Easy-to-view logs of all credits and debits.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Backend/Database:** Supabase (PostgreSQL)
* **Hosting:** Vercel (Auto-deployed via GitHub)
